# ポリテクメモリアル

ポリテクセンターを舞台にした学習型ビジュアルノベルゲームです。
A learning-based visual novel game set in a Polytech Center.

---

## 遊び方 (日本語)

### 🌐 Web版（推奨・インストール不要）

**ブラウザで今すぐ遊べます — インストール不要。**

1. 以下のリンクをクリックしてゲームを開きます。
   **[ゲームを開く](https://mp9999-git.github.io/polytech2025/)**

2. ローディング画面が終わったら、画面をタップ／クリックしてスタートします。

3. 訓練生の名前と先生の名前を入力してゲームを開始してください。

4. ストーリーを読み進め、3択クイズに挑戦しながら全5ステージをクリアしましょう。
   （Network → PLC → Database → Java → Android）

---

### 💻 PC版（ローカル起動）

インターネット接続なしでも遊べます。このリポジトリをダウンロードし、以下の手順で起動してください。

**動作に必要なもの:** Windows PC（PowerShell が使えること）

1. リポジトリを ZIP でダウンロードするか `git clone` します。
2. フォルダ内の **`PCでゲームを起動.bat`** をダブルクリックします。
3. ローカルサーバーが自動起動し、ブラウザでゲームが開きます。
4. ゲームを終了するには、起動した黒いウィンドウを閉じてください。

> **注意:** ブラウザで直接 `index.html` を開いても動作しません（セキュリティ制限のため）。
> 必ず `PCでゲームを起動.bat` 経由で起動してください。

## How to Play (English)

### 🌐 Web Version (Recommended — no installation)

**Play instantly in your browser — no installation required.**

1. Click the link below to open the game.
   **[Play Now](https://mp9999-git.github.io/polytech2025/)**

2. Wait for the loading screen to finish, then tap or click to start.

3. Enter your trainee name and teacher names to begin.

4. Read the story and challenge 3-choice quizzes across all 5 stages.
   (Network → PLC → Database → Java → Android)

---

### 💻 PC Local Version

You can also play offline by running the game locally.

**Requirement:** Windows PC with PowerShell (included by default on Windows 7+)

1. Download the repository as a ZIP or `git clone` it.
2. Double-click **`PCでゲームを起動.bat`** inside the folder.
3. A local server starts automatically and the game opens in your browser.
4. To quit, close the black server window.

> **Note:** Opening `index.html` directly in a browser will not work due to browser security restrictions.
> Always launch via `PCでゲームを起動.bat`.

---

|----------|---------|

| PC (Local) | Windows 7+, PowerShell pre-installed |

> Android: Browser menu → "Add to Home Screen"

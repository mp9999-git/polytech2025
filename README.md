# ポリテクメモリアル

ポリテクセンターを舞台にした学習型ビジュアルノベルゲームです。
A learning-based visual novel game set in a Polytech Center.

---

## 遊び方 (日本語)

**ブラウザで今すぐ遊べます — インストール不要。**

1. 以下のリンクをクリックしてゲームを開きます。
   **[ゲームを開く](https://mp9999-git.github.io/polytech2025/)**

2. ローディング画面が終わったら、画面をタップ／クリックしてスタートします。

3. 訓練生の名前と先生の名前を入力してゲームを開始してください。

4. ストーリーを読み進め、3択クイズに挑戦しながら全5ステージをクリアしましょう。
   （Network → PLC → Database → Java → Android）

### 推奨環境
| 環境 | 詳細 |
|------|------|
| PC   | Chrome / Edge / Firefox（最新版）、横画面推奨 |
| スマートフォン・タブレット | Chrome / Safari（最新版）、横向き固定 |

> **ヒント:** スマホでホーム画面に追加すると、アプリのようにフルスクリーンで起動できます。
> iOS: 共有メニュー →「ホーム画面に追加」
> Android: ブラウザメニュー → 「ホーム画面に追加」

---

## How to Play (English)

**Play instantly in your browser — no installation required.**

1. Click the link below to open the game.
   **[Play Now](https://mp9999-git.github.io/polytech2025/)**

2. Wait for the loading screen to finish, then tap or click to start.

3. Enter your trainee name and teacher names to begin.

4. Read the story and challenge 3-choice quizzes across all 5 stages.
   (Network → PLC → Database → Java → Android)

### Recommended Environment
| Platform | Details |
|----------|---------|
| PC       | Chrome / Edge / Firefox (latest), landscape orientation recommended |
| Mobile / Tablet | Chrome / Safari (latest), landscape orientation |

> **Tip:** On mobile, add to your home screen to launch in fullscreen like a native app.
> iOS: Share menu → "Add to Home Screen"
> Android: Browser menu → "Add to Home Screen"
